abp集成ant blazor

2020-11-12

引入Ant Design of Blazor包

 Install-Package AntDesign -Version 0.4.0-nightly-2011120611
 Install-Package AntDesign.Pro.Layout -Version 0.1.0-nightly-201103141915
 Install-Package AntDesign.Charts -Version 0.2.0

注册Ant Design of Blazor服务

1.注册服务

在MyAbpBlazorModule文件中ConfigureServices方法加入以下代码

  context.Services.AddAntDesign();

2.添加命名空间

在_Imports.razor文件中加入以下代码

  @using AntDesign
  @using AntDesign.Charts
  @using AntDesign.Pro.Layout

添加Ant Design of Blazor组件

1.添加css

在wwwroot文件中的index.html文件加入以下代码

 <link href="_content/AntDesign/css/ant-design-blazor.css" rel="stylesheet" />
 <link href="_content/AntDesign.Pro.Layout/css/ant-design-pro-layout-blazor.css" rel="stylesheet" />

2.添加js

 <script src="_content/AntDesign.Charts/g2plot.js"></script>
 <script src="_content/AntDesign/js/ant-design-blazor.js"></script>
 <script src="_content/AntDesign.Charts/ant-design-charts-blazor.js"></script>

卸载默认主题

在Abp的Blzaor项目默认使用的主题是Volo.Abp.AspNetCore.Components.WebAssembly.BasicTheme.Themes.Basic导入风格和布局和Ant Design of Blazor有差异，最终 我选择的将其卸载掉不用并重写自定义布局界面。

注意：暂时不清楚项目在后期是否出错，所以我按照默认主题的路径和命名进行重写。

注意：卸载掉主题后请在MyAbpBlazorModule移除相关代码

Uninstall-Package Volo.Abp.AspNetCore.Mvc.UI.Theme.Basic

1.添加MainLayout.razor文件并输入如下代码

@inherits LayoutComponentBase
@using Volo.Abp.Ui.Branding
@inject IBrandingProvider BrandingProvider
<AntDesign.Pro.Layout.BasicLayout Logo="@("https://gw.alipayobjects.com/zos/rmsportal/KDpgvguMpGfqaHPjicRK.svg")" MenuData="_menuData">
    <RightContentRender>

    </RightContentRender>
    <ChildContent>
        @Body
    </ChildContent>
</AntDesign.Pro.Layout.BasicLayout>
@code
{
    private MenuDataItem[] _menuData = { };

    protected override async Task OnInitializedAsync()
    {
        await base.OnInitializedAsync();
        _menuData = new MenuDataItem[3];
        _menuData[0] = new MenuDataItem() {
            Path= "/menu1",
            Name= "菜单1",
            Key= "menu1",
            Icon= "global"
        };

        _menuData[1] = new MenuDataItem()
        {
            Path = "/menu2",
            Name = "菜单2",
            Key = "menu2",
            Icon = "table"
        };

        _menuData[2] = new MenuDataItem()
        {
            Path = "/menu3",
            Name = "菜单3",
            Key = "menu3",
            Icon = "user"
        };
    }
}

2.添加App.razor文件并输入如下代码

@using Volo.Abp.AspNetCore.Components.WebAssembly.Theming.Routing
@using Microsoft.Extensions.Options
@inject IOptions<AbpRouterOptions> RouterOptions
<CascadingAuthenticationState>
    <Router AppAssembly="RouterOptions.Value.AppAssembly"
            AdditionalAssemblies="RouterOptions.Value.AdditionalAssemblies">
        <Found Context="routeData">
            <AuthorizeRouteView RouteData="@routeData" DefaultLayout="@typeof(MainLayout)">
                <NotAuthorized>
                    @if (!context.User.Identity.IsAuthenticated)
                    {
                        <RedirectToLogin />
                    }
                    else
                    {
                        <p>You are not authorized to access this resource.</p>
                    }
                </NotAuthorized>
            </AuthorizeRouteView>
        </Found>
        <NotFound>
            <LayoutView Layout="@typeof(MainLayout)">
                <p>Sorry, there's nothing at this address.</p>
            </LayoutView>
        </NotFound>
    </Router>
</CascadingAuthenticationState>

3.添加RedirectToLogin.razor文件并输入如下代码

@inject NavigationManager Navigation
@using Microsoft.AspNetCore.Components.WebAssembly.Authentication
@code {
    protected override void OnInitialized()
    {
        Navigation.NavigateTo($"authentication/login?returnUrl={Uri.EscapeDataString(Navigation.Uri)}");
    }
}
