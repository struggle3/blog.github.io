1. 环境要求

  1. JDK版本：1.8版本
  2. Spring版本：5.X版本
  3. Maven版：3.X版本以上

2. 新建项目

2.1. 创建Maven项目

2.2. 设置坐标

2.3. 设置Maven

3. 修改Maven依赖pom.xml

3.1. 修改JDK版本

  <properties>
    <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    <maven.compiler.source>1.8</maven.compiler.source>
    <maven.compiler.target>1.8</maven.compiler.target>
  </properties>

3.2. pluginManagement移除（可选）

删除build标签下面的pluginManagement所有内容

3.2 junit移除（可选）

删除dependencies标签下面的junit依赖

3.3. 添加spring框架依赖

<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-context</artifactId>
    <version>5.2.13.RELEASE</version>
</dependency>

3.4. 定义接口与实现类

public interface NewsItemService {
    void ShowTitle();
}

public class NewsItemServiceImpl implements NewsItemService {
    public NewsItemServiceImpl() {
    super();
        System.out.println("无参数构造方法");
    }
    @Override
    public void ShowTitle() {
        System.out.println("业务方法");
    }
}

4. 创建Spring配置文件

在src/main/resources/目录现创建一个xml文件，文件名可以随意，但Spring建议的名称为applicationContext.xml。

spring配置中需要加入约束文件才能正常使用，约束文件是xsd扩展名。

4.1. 添加resources文件夹

4.1. 添加applicationContext.xml文件

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">


<!--    <bean />：用于定义一个实例对象。一个实例对应一个 bean元素。-->
<!--    id：该属性是Bean实例的唯一标识，程序通过id属性访问Bean，Bean 与 Bean 间的依赖关系也是通过id 性关联-->
<!--    class：指定该Bean所属的类，注意这里只能是类，不能是接口-->

    <bean id="newsItemServiceImpl" class="NewsItemServiceImpl"></bean>
</beans>

5. 测试

在test目录上的AppTest类中添加testShowTitle方法进行测试

package org.example;

import static org.junit.Assert.assertTrue;

import org.junit.Test;
import org.springframework.context.ApplicationContext;
import org.springframework.context.support.ClassPathXmlApplicationContext;

/**
 * Unit test for simple App.
 */
public class AppTest 
{
    /**
     * Rigorous Test :-)
     */
    @Test
    public void shouldAnswerWithTrue()
    {
        assertTrue( true );
    }

    @Test
    public  void testShowTitle()
    {
        ApplicationContext ctx=new ClassPathXmlApplicationContext("applicationContext.xml");
        NewsItemServiceImpl news=ctx.getBean("newsItemServiceImpl",NewsItemServiceImpl.class);
        news.showTitle();
    }
}
