Spring 基于XML依赖注入

2021-03-04

bean实例在调用无参构造器创建对象后对bean对象的属性进行初始化。初始化是由容器自动完成的称为注入。

根据注入有三类：setter注入、构造器注入、自动注入（引用类型）。

1. 构造器注入

构造器注入是指在构造调用者实例的同时，完成被调用者的实例化。

标签中用于指定参数的属性有：

  • name：指定参数名称。

  • index：指明该参数对应着构造器的索引位置，从0开始保证赋值顺序要与构造器中的参数顺序一致。

dao类代码

package org.example.dao;

public class NewsItemDao {
    public  void  showTitle()
    {
        System.out.println("dao类方法执行。");
    }
}

service类代码

package org.example.service.impl;

import org.example.dao.NewsItemDao;
import org.example.service.NewsItemService;

public class NewsItemServiceImpl implements NewsItemService {

    private NewsItemDao newsItemDao;

    public NewsItemServiceImpl(NewsItemDao newsItemDao) {
        this.newsItemDao = newsItemDao;
        System.out.println("带参数构造方法");
    }

    @Override
    public void showTitle() {
        System.out.println("业务方法");
    }
}

bean配置

<bean id="newsItemDao" class="org.example.dao.NewsItemDao"></bean>
<bean id="newsItemServiceImpl" class="org.example.service.impl.NewsItemServiceImpl">
    <constructor-arg name="newsItemDao" ref="newsItemDao"></constructor-arg>
</bean>

junit代码

@Test
public  void testShowTitle()
{
    ApplicationContext ctx=new ClassPathXmlApplicationContext("applicationContext.xml");
    NewsItemServiceImpl news=ctx.getBean("newsItemServiceImpl",NewsItemServiceImpl.class);
    news.showTitle();
}

2. Setter注入

set注入是指通过setter方法传入被调用者的实例引用类型使用ref进行对象id指定，基本类型使用value指定。

dao类代码

package org.example.dao;

public class NewsItemDao {
    public  void  showTitle()
    {
        System.out.println("dao类方法执行。");
    }
}

service类代码

package org.example.service.impl;

import org.example.dao.NewsItemDao;
import org.example.service.NewsItemService;

public class NewsItemServiceImpl implements NewsItemService {

    private String Title;
    public void setTitle(String title) {
        System.out.println("title注入方法执行。"+title);
        Title = title;
    }

    private NewsItemDao newsItemDao;
    public void setNewsItemDao(NewsItemDao newsItemDao) {
        System.out.println("newsItemDao注入方法执行。");
        this.newsItemDao = newsItemDao;
    }

    @Override
    public void showTitle() {
        newsItemDao.showTitle();
        System.out.println("service方法执行。");
    }
}

bean配置

<bean id="newsItemDao" class="org.example.dao.NewsItemDao"></bean>
<bean id="newsItemServiceImpl" class="org.example.service.impl.NewsItemServiceImpl">
    <property name="newsItemDao" ref="newsItemDao"/>
    <property name="title" value="mytitle"/>
</bean>

junit代码

@Test
public  void testShowTitle()
{
    ApplicationContext ctx=new ClassPathXmlApplicationContext("applicationContext.xml");
    NewsItemServiceImpl news=ctx.getBean("newsItemServiceImpl",NewsItemServiceImpl.class);
    news.showTitle();
}

3. 自动注入

对于引用类型属性的注入，可以不通过xml文件配置显示注入则通过标签设置autowire属性值为引用类型属性进行隐式自动注入；可以分为两种：

  • byName：根据名称自动注入
  • byType：根据类型自动注入

3.1. byName

xml文件中被调用者bean的id值与代码中调用者bean类的属性名相同时可使用byName方式让容器自动将被调用者bean注入给调用者bean。

容器是通过调用者的bean类的属性名与配置文件的被调用者bean的id进行比较而实现自动注入的。

dao类代码

package org.example.dao;

public class NewsItemDao {
    public  void  showTitle()
    {
        System.out.println("dao类方法执行。");
    }
}

service类代码

package org.example.service.impl;

import org.example.dao.NewsItemDao;
import org.example.service.NewsItemService;

public class NewsItemServiceImpl implements NewsItemService {

    private NewsItemDao newsItemDao;

    public void setNewsItemDao(NewsItemDao newsItemDao) {
        this.newsItemDao = newsItemDao;
    }

    @Override
    public void showTitle() {
        newsItemDao.showTitle();
        System.out.println("service方法执行。");
    }
}

bean配置

<bean id="newsItemDao" class="org.example.dao.NewsItemDao"></bean>
<!-- autowire="byName"会从容器中查找与实体类的域属性同名的Bean 的id，并将该Bean对象自动注入给该域属性 -->
<bean id="newsItemServiceImpl" class="org.example.service.impl.NewsItemServiceImpl" autowire="byName">
</bean>

junit代码

@Test
public  void testShowTitle()
{
    ApplicationContext ctx=new ClassPathXmlApplicationContext("applicationContext.xml");
    NewsItemServiceImpl news=ctx.getBean("newsItemServiceImpl",NewsItemServiceImpl.class);
    news.showTitle();
}

3.1. byType

byType方式自动注入要求xml文件中被调用者bean的class属性指定的类型要与代码中调用者bean类的引用类型属性同源，满足is-a关系（子类或是实现类）。

bean配置

<bean id="newsItemDao" class="org.example.dao.NewsItemDao"></bean>
<!-- autowire="byType"会从容器中查找与实体类的域属性类型具有 is-a 关系的Bean，并将该Bean对象自动注入给该域属性 -->
<bean id="newsItemServiceImpl" class="org.example.service.impl.NewsItemServiceImpl" autowire="byName">
</bean>
