Spring Boot简介

2021-03-11

1. 介绍

通过Spring Boot可以很方便的构建一个独立的web应用程序，Spring Boot提供了开箱即用的基础设置，开发者不用关心过多的配置，可以快速开启一个Web容器进行应用程序开发。

Spring Boot是一个框架，一种全新的Spring编程规范，它的出现简化了框架的使用，具体表现在则在Spring众多框架中所需的大量的配置文件，所以Spring Boot是一个服务于框架的框架，服务范围是简化配置文件。

1.1. 特性

  1. 独立运行Spring项目：Spring Boot可以以jar包的形式独立运行。
  2. 内嵌Servlet容器：无须war包形式部署项目(内嵌Tomcat、Jetty、Undertow)。
  3. 提供starter依赖：提供基础POM文件来简化Maven配置。
  4. 自动配置Spring：根据项目依赖来自动配置Spring 框架。
  5. 基础生产的应用监控：提供http,sh,telnet等。
  6. 无生成代码和xml配置：不需要xml配置即可完成Spring配置。

2. 项目结构说明：

  1. .mvn：maven相关内容，如果电脑中不环境可以用maven进行编译。
  2. src:源代码目录
  3. mvnw:提供Linux的相关shell脚本
  4. mvnw.cmd:提供Windows的相关bat脚本
  5. pom.xml:项目包管理文件

3. pom.xml说明

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
<!--    springboot父级项目依赖-->
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>2.4.5</version>
        <relativePath/> <!-- lookup parent from repository -->
    </parent>
    <groupId>com.springboot01</groupId>
    <artifactId>springboot01</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <name>springboot01</name>
    <description>Demo project for Spring Boot</description>
<!--    项目属性配置 -->
    <properties>
        <java.version>1.8</java.version>
    </properties>
    <dependencies>
<!--        web项目依赖-->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

<!--        测试依赖-->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <build>
        <plugins>
<!--            springboot提供的项目编译插件-->
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
    </build>

</project>
