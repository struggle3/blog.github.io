Spring MVC

2021-03-17

1 MVC介绍

MVC成为计算机科学领域重要的构建模式已经有多年历史，1979年最初被命令为事物-模型-视图-编辑器，后来简化成了模型 -视图-控制器。

MVC是一种强大而简洁的方式，尤其是应用在Web应用程序中。虽然关注点的显式分享稍微增加了应用程序设计的复杂性，但是MVC带来益处要远远超过它所带来的不便。自从此入以来，MVC已经在数十种框架中应用。

MVC将应用程序的用户界面分为三个主要部分：

  1. 模型(Model)：应用程序数据以及操作数据的规则。
  2. 视图(View)：应用程序界面显示方式。
  3. 控制器(Controller)：应用程序数据的处理逻辑。

1.3. Spring MVC

Spring MVC是Spring Framework的一部分是基于Java实现MVC的轻量级Web框架具有角色划分清晰，分工明细，配置强大并且和Spring框架无缝结合，Spring2.5以后开发者可以采用基于声明方式注解的方式进行Controller开发，通过DispatcherServlet将请求分发到不同的处理器。

Spring MVC的特点：

  1. 轻量
  2. 高效
  3. 兼容Spring
  4. 功能强大:RESTful、数据验证、格式化、绑定机制、本地化、主题等
  5. 简洁灵活
