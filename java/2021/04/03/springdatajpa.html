2. JPQL方式

JPQL语法和SQL类似都可以实现对数据库中表结构里面的数据是持久化操作，但二者不的地方在于JPQL是操作对象和属性，SQL操作的表和字段。

使用JPQL必须满足以下要求：

  1. 在Repository接口中定义方法;
  2. 通过注解的形式配置JPQL语句;
  3. @Query注解表示查询;value参数值是JPQL语句,nativeQuery参数值是false表示是jpql查询方式，true表示是sql查询方式；
  4. ”?”表示查询条件占位符默认顺序和方法参数名一致”?”索引的方式手动指定占位符顺序；
  5. 插入和更新操作使用@Modifying注解声明方法为更新;
  6. 更新操作必须手动添加@Transactional和@Rollback(value = false)注解启用事务;
  7. 使用原生SQL查询返回值必须是List<Object []>类型。

2.1. Repository代码

package com.dao;

import com.domain.User;
import org.springframework.data.jpa.repository.JpaRepository;
import org.springframework.data.jpa.repository.JpaSpecificationExecutor;

/**
 * JpaRepository<实休，主键>
 * 封装了基本CRUD操作
 * JpaSpecificationExecutor<实体>
 * 封装了复杂查询（分页）
 */
public interface UserDao extends JpaRepository<User,Integer>, JpaSpecificationExecutor<User> {
    /*
    * 查询
    * */
    @Query(value="from User where name = ?")
    public User select(String name);
    /*
    * 更新
    * */
    @Query(value = " update User set name = ?2 where id = ?1 ")
    @Modifying
    public void update(Integer id,String name);
    /*
    * 插入
    * */
    @Modifying
    @Query(value = "insert into Users(name, userName) values(?1, ?2)",nativeQuery = true)
    public  void insert(String name,String userName);
    /*
    * 删除
    * */
    @Modifying
    @Query(value = "delete User where id= ?")
    public  void delete(Integer id);

}

3. 方法名约定查询

总结一下：查询功能的方法名由查询策略（关键字）、查询字段和一些限制性条件组成。
