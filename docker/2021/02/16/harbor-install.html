harbor安装

2021-02-16

1. 必备条件

  • Python应该是2.7或更高版本
  • Docker引擎应为1.10或更高版本
  • DockerCompose需要为1.6.0或更高版本

2. harbor安装

2.1. 官方地址：

  • https://github.com/vmware/harbor/releases

2.2. 修改harbor.cfg

  • hostname：目标的主机名或者完全限定域名

  • ui_url_protocol：http或https。默认为http

  • db_password：用于db_auth的MySQL数据库的根密码。更改此密码进行任何生产用途

  • max_job_workers：（默认值为3）允许工作人员同步到远程目标,增加此数字允许系统中更多的并发复制作业。

  • customize_crt：（on或off。默认为on）当此属性打开时prepare脚本将为注册表的令牌的生成/验证创 建私钥和根证书

  • ssl_cert：SSL证书的路径，仅当协议设置为https时才应用

  • ssl_cert_key：SSL密钥的路径，仅当协议设置为https时才应用

  • secretkey_path：用于在复制策略中加密或解密远程注册表的密码的密钥路径

3. 安装步骤

3.1 创建https证书以及配置相关目录权限

  • openssl genrsa -des3 -out server.key 2048 #生成2048位的私钥
  • openssl req -new -key server.key -out server.csr #生成证书
  • cp server.key server.key.org #备份私钥
  • openssl rsa -in server.key.org -out server.key #退密钥保证nginx使用
  • openssl x509 -req -days 365 -in server.csr -signkey server.key -out server.crt
  • mkdir -p /data/cert
  • mv server.* /data/cert
  • chmod -R 777 /data/cert #说明

3.2 复制harbor

  • 直接使用xftp上传到/usr/local/目录中
  • 解压tar -zxvf harbor-offline-installer-.tgz
  • 进入/usr/local/harbor
  • 修改harbor.cfg相关参数
  • 执行./install.sh

4. 使用

4.1. 服务端

4.1.1. 指定镜像仓库地址

添加/etc/docker/daemon.json配置以便于docker认证通过

{
    "insecure-registries": ["ip"]
}

4.1.2. 下载测试镜像

docker pull hello-world

4.1.3. 给镜像重新打标签

dockertag hello-world serverip/hello-world:latest

4.1.4. 登录进行上传

dockerlogin serverip

4.2. docker客户端

4.2.1. 指定镜像仓库地址

添加/etc/docker/daemon.json配置以便于docker认证通过

{
    "insecure-registries": ["ip"]
}

4.2.2. 下载测试镜像

docker pull ip/hello-world:latest

注意:以上环境如果没有DNS解析服务必须要在/etc/hosts文件中添加一行解析记录
