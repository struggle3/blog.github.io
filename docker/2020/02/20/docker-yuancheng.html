docker远程

2020-02-20

docker同样可以以远程的方式进行访问，但存在安全风险在生产环境中不建议这样使用。

配置远程访问1 修改/etc/docerk/daemon.jso文件

"hosts":["tcp://0.0.0.0:2375","unix://var/run/docker.sock"]

配置远程访问2 修改/usr/lib/systemd/system/docker.service

ExecStart=/usr/bin/dockerd -H tcp://0.0.0.0:2375 -H unix://var/run/docker.sock

远程命令

docker -H ip:port command
