docker安装

2020-02-12

基于Linux的docker的安装方法

1. 脚本安装

直接使用官方给的安装脚本进行安装，脚本装安装最新版本docker容器，在生产环境中不建议使用此方法因为最新版可能存在一些bug导致一些异常问题。

yum update
$ curl -sSL https://get.docker.com/ | sh

2. yum安装

yum update
cat >/etc/yum.repos.d/docker.repo <<-EOF
[dockerrepo]
name=Docker Repository 
baseurl=https://yum.dockerproject.org/repo/main/centos/7 
enabled=1
gpgcheck=1 
gpgkey=https://yum.dockerproject.org/gpg EOF 
yum install docker

3. rpm安装（推荐）

3.1. 下载地址:

https://download.docker.com/linux/centos/7/x86_64/stable/Packages/

3.2. 步骤:

  1. 下载docker-ce-17.03.0.ce-1.el7.centos.x86_64.rpm文件
  2. 下载docker-ce-selinux-17.03.0.ce-1.el7.centos.noarch.rpm文件
  3. 在root下创建docker目录
  4. 移动(mv docker* docker)以上两个文件到docker目录
  5. 进入docker(cd docker)目录执行yum -y install *命令

3.3. docker测试

执行下面命令

systemctl start docker 
systemctl status docker 
systemctl enable docker
docker run hello-world

3.4. 防火墙设置（可选）

因为docker会利用防火墙进行一些设置。

systemctl stop firewalld #关闭防火墙
systemctl disable firewalld #禁用开机自启
getenforce #查看selinux是否为Disabled
yum -y install iptables-services
systemctl start iptables #启动
systemctl enable iptables #启用开机自启
iptables -F #清空规则
service iptables save #保存
yum update
