使用ubuntu做为dotnet core开发环境

2019-04-27

一、安装google浏览器

1、下载安装包(传送门：http://www.google.cn/intl/zh-CN/chrome/browser/desktop/index.html)
2、使用sudo dpkg-i 安装名称

二、安装nodejs

1、下载官方最新
2、解包:tar -zxvf xxx.tar.zx
3、设置node和npm软链接
      sudo ln -s 安装位置 /usr/local/bin/node
      sudo ln -s 安装位置 /usr/local/bin/npm

三、安装angular/cli

1、命令：sudo npm install -g @angular/cli
2、设置软链接： sudo ln -s 安装位置 /usr/local/bin/ng

四、安装git

1、添加git源：sudo add-apt-repository ppa:git-core/ppa
2、更新源列表：sudo apt update
3、安装git：sudo apt install git

五、安装vscod

1、下载deb安装包(传送门：https://code.visualstudio.com/Download)
2、使用sudo dpkg -i 安装名称

六、安装dotnet core

1、按官网步骤进行即可（传送门：https://dotnet.microsoft.com/download/linux-package-manager/ubuntu16-04/runtime-current）

七、Ubuntu 16.04 和 windows 时间不同步的问题

1、每次进入Ubuntu 16.04后再回到windows，就会发现时间对不上，少了8小时，这是因为Ubuntu 16.04的时区设置不正确造成的，解决方法是执行下面的命令：sudo timedatectl set-local-rtc 1
